Hello! You've found Pluto's "dynos" folder.

This is the place you can put all downloadable content, including models,
color codes, animations, and vanilla eye textures.


MODEL PACKS
-----------

Place into `packs/` and extract! Your final folder structure should look like:
`dynos/packs/MyModelFolder/mario_geo.bin`


ANIMATIONS
----------

Currently, Pluto only accepts .panim animations.
If you drop a .json into the `anims/` folder, it will be automatically converted
to a PlutoAnim -- but don't worry, your .json won't be deleted.


COLORCODES
----------

All color code (GameShark codes) are saved to either .gs or .txt - Pluto can read
them both.
You can also create, import, copy, or save GS codes from within the editor.


EYES
----

The `eyes/` folder is the place for *all* of your "vanilla" eye .png files.
Most Mario-style models will reuse this same eye texture folder. So you should
put all of them in here!
Subfolders are also accepted.
